# Secure Vault

## What it does (simple English)
Secure Vault is a full-stack web application designed for the safe storage and management of sensitive data. It allows users to securely register, log in, and manage their private information in a protected environment.

## Tech stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (MERN Stack)

## Features
- Secure user authentication and authorization
- Protected client-server architecture
- Responsive dashboard for managing stored data
- RESTful API integration

## Challenges I faced
- Setting up secure authentication tokens and ensuring data moving between the client and server remained protected.
- Structuring the backend routes to handle complex database queries efficiently.

## What I learned
- I gained practical experience building a complete MERN stack application from scratch.
- I learned best practices for securing API endpoints and managing state in a React frontend.

## How to run
1. Clone the repo: `git clone https://github.com/kab5DeR4/Secure_Vault.git`
2. Setup the backend:
   - `cd server`
   - `npm install`
   - `npm start`
3. Setup the frontend:
   - `cd ../client`
   - `npm install`
   - `npm start`
